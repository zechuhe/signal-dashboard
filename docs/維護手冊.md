## 1. 部署

### 方法 A：本地（推薦開發用）
```
雙擊 start.bat
```
- 自動偵測 Python 路徑（優先 Antigravity Toolset，fallback PATH）
- 自動偵測可用 port（8443 → 9443 → 8080 → ...）
- 自動開啟瀏覽器

### 方法 B：GitHub Pages（推薦展示用）
```bash
git push origin master
```
GitHub Pages 自動部署（1-2 分鐘生效）。

### 方法 C：任意 HTTP Server
```bash
# Node.js
npx serve .

# Python
python -m http.server 8443

# PHP
php -S localhost:8443
```

## 2. Config 設定

### config/app.json（前端 UI）

| 設定 | 路徑 | 說明 |
|------|------|------|
| Port | `app.port` | HTTP server port |
| 事件分類 | `categories.*` | 名稱/顏色/標記 |
| 影響力樣式 | `impact.*` | 利好/利空/中性的顏色 |
| MA 均線週期 | `chart.indicators.MA.periods` | 預設 [5,10,20,60] |
| 蠟燭顏色 | `chart.candle.*` | 漲(upColor)/跌(downColor) |
| 搜尋結果數 | `search.maxResults` | 預設 15 |
| 交易所 | `exchanges.*` | 市場定義 + Yahoo 後綴 |
| CDN | `cdn.klinecharts` | KLineChart 版本 |

### config/sources.json（後端 API）

| 設定 | 路徑 | 說明 |
|------|------|------|
| HKEX | `sources.hkex.endpoints.*` | 港交所公告 API |
| Google | `sources.google_news.endpoints.*` | Google News RSS |
| 東方財富 | `sources.eastmoney.endpoints.*` | 搜索 API |
| 請求延遲 | `fetch_settings.request_delay_sec` | API 調用間隔 |
| 超時 | `fetch_settings.timeout_sec` | 請求超時 |

## 3. 數據管理

### 股價數據更新
```bash
# 修改 scripts/fetch_prices.py 的 STOCKS dict 加入目標股票，然後：
python scripts/fetch_prices.py
```

### 公開新聞撈取
```bash
python scripts/fetch_news.py --stock 0700 --days 30 --output data/public_events_0700.json
```

### LLM 影響力分析
```bash
# 需設定 GEMINI_API_KEY 環境變數或 .env 檔案
python scripts/analyze_impact.py --events data/events.json --prices data/price_HKEX_0001.json --llm --output data/analysis.json
```

### 手動數據管理
- 管理面板（齒輪按鈕）→ 事件管理 tab → CRUD
- 管理面板 → 數據管理 → 匯出全部 / 清除

## 4. 異常處理

| 問題 | 原因 | 解決 |
|------|------|------|
| 頁面空白 | file:// 協議 | 必須用 HTTP server，不能雙擊 html |
| start.bat 找不到 Python | PATH 沒有 python | 編輯 start.bat 的 PYTHON_PATH |
| 搜尋無結果 | stock_list.json 未載入 | 確認 data/stock_list.json 存在 |
| 股票打開後無圖表 | 缺 price JSON | 用 fetch_prices.py 拉取 |
| 事件標記不顯示 | 事件日期不在股價範圍內 | 確認事件日期在股價 data range 內 |
| 管理面板數據源顯示錯誤 | sources.json 格式問題 | 驗證 JSON 格式 |

## 5. 數據備份

- **用戶事件**：存在瀏覽器 localStorage，換瀏覽器/清快取會遺失。定期用「匯出」按鈕備份。
- **股價數據**：靜態 JSON，隨 Git 版控。
- **config**：隨 Git 版控。`.env` 不入版控（在 .gitignore）。

## 6. 版號規則

- 格式：`vMajor.Minor`
- Major：架構重構或圖表引擎更換
- Minor：功能新增或 bug 修復
- 每次發版更新：CHANGELOG.md（根目錄）+ config/app.json `app.version`
